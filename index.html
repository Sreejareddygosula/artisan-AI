Artisan AI Assistant

Flask + Google Gemini app to help artisans create authentic stories, descriptions, captions, hashtags, and ad copy. Optional image analysis powered by Gemini Vision.

Open App (Localhost)

How to run locally

  1. Python 3.10+ recommended
  2. In PowerShell: 
    python -m venv .venv
. .\.venv\Scripts\Activate.ps1
pip install -r requirements.txt
  3. Copy .env.example to .env and set GOOGLE_API_KEY
  4. Start the server: 
    python app.py
  5. Open: http://localhost:5000

Note: This static index.html is only a landing page. The live app UI is rendered from templates/index.html by Flask.

API Endpoints

  • POST /api/generate – Generate text (story, description, captions, hashtags, ad copy)
  • POST /api/analyze-image – Upload image + optional fields to extract attributes and suggest captions

Cloud Run (optional)

Use the included Dockerfile and follow the steps in README.md to deploy.
